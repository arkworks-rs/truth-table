# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

The Code of Conduct applies within all project spaces, and it also applies when an individual is officially representing the project or its community in public spaces.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at:

- alrshir@seas.upenn.edu

All complaints will be reviewed and investigated promptly and fairly. The maintainers are obligated to respect the privacy and security of the reporter of any incident.
